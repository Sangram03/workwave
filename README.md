# WorkWave

WorkWave is a VS Code extension designed to improve developer productivity by providing quick access to common development tools and workflows.

## Features

* Streamlined development workflow
* Easy-to-use commands
* Productivity-focused utilities
* Lightweight and fast

## Usage

1. Install the extension.
2. Open the Command Palette (`Ctrl+Shift+P`).
3. Search for WorkWave commands.
4. Execute the desired action.

## Requirements

* Visual Studio Code 1.80.0 or later
